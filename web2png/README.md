## web2png

Take screenshot of a web page and save it into a png file.

## usage

```
Usage: web2png [OPTIONS] <PATH>

Arguments:
  <PATH>

Options:
      --output <OUTPUT>  [default: image.png]
      --width <WIDTH>    [default: 1920]
      --height <HEIGHT>  [default: 1080]
  -h, --help             Print help information
```
